Madhav Ecommerce Dashboard – Power BI Project

Overview

The Madhav Ecommerce Dashboard is an interactive Power BI project designed to analyze ecommerce sales performance, customer behavior, revenue trends, and product insights through dynamic visualizations and KPIs.

• Revenue and profit tracking
• Customer purchase insights

• Interactive filters and slicers
• KPI tracking dashboard
