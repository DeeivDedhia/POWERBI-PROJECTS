Blinkit Dashboard – Power BI Project

Overview

The Blinkit Dashboard is an interactive Power BI project designed to analyze grocery delivery sales, customer trends, order performance, and operational KPIs through dynamic visualizations.

Features

• Grocery sales analysis
• Order performance tracking
• Customer insights dashboard
• Interactive filters and reports
• KPI monitoring
• Trend analysis and visualization
