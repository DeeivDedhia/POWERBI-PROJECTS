Loan Dashboard – Power BI Project

Overview

The Loan Dashboard is an interactive Power BI project designed to analyze loan applications, approval trends, customer profiles, and financial insights through dynamic dashboards and KPIs.

Features

• Loan application analysis
• Approval and rejection tracking
• Customer profile insights
• Financial KPI monitoring
• Interactive filters and slicers
• Trend analysis and reporting

Tools & Technologies Used

• Power BI
• Power Query
• DAX
• Data Modeling
• Excel / CSV datasets

• Customer Insights
