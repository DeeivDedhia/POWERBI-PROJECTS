# Election Analysis Dashboard (Power BI)

Interactive Power BI dashboard analyzing election results across states, constituencies, parties, and candidates.

## Features
- State-wise election analysis
- Party performance tracking
- Winner vs runner-up vote share comparison
- Constituency-level insights
- Interactive filters and slicers
- KPI cards and visual analytics

## Tools Used
- Power BI
- Power Query
- DAX
- CSV Datasets

## Key Insights
- Analyzed vote share trends across multiple states
- Compared party-wise performance
- Identified winning margins and runner-up statistics
- Built dynamic visual dashboards for data storytelling
