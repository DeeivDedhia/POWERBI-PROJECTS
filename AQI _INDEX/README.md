AQI Index Dashboard – Power BI Project

Overview

The AQI Index Dashboard is an interactive Power BI project designed to analyze air quality data, pollution levels, AQI trends, and environmental insights using dynamic visualizations and KPIs.

Features

• Air quality analysis
• AQI trend visualization
• Pollution level monitoring
• Interactive charts and reports
• Dynamic filters and slicers
• Environmental KPI tracking

Tools & Technologies Used

• Power BI
• Power Query
• DAX
• Data Modeling
• Excel / CSV datasets

Dashboard Highlights

• AQI Overview
• Pollution Trend Analysis
• Environmental Insights
• KPI Cards
• Comparative Analysis
• Interactive Reports
