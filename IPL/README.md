# 🏏 IPL Data Analysis Dashboard – Power BI Project

An interactive Power BI dashboard built to analyze Indian Premier League (IPL) data with insights into teams, players, matches, wins, and tournament performance.

------------------------------------------------------------

📌 Project Overview

This project provides visual analysis of IPL data using Power BI.

The dashboard helps users explore:
• Team performances
• Match statistics
• Winning trends
• Player analysis
• Venue insights
• Season-wise comparisons

The main objective of this project is to convert raw IPL datasets into meaningful insights using data visualization and analytics.

------------------------------------------------------------

🚀 Features

• Interactive Power BI Dashboard
• Team-wise performance analysis
• Top players and match winners
• Season-wise trends
• Venue-based insights
• Dynamic filters and slicers
• KPI cards for quick analytics

------------------------------------------------------------

🛠 Tools & Technologies Used

• Power BI
• DAX (Data Analysis Expressions)
• Power Query
• Data Modeling
• CSV Dataset

------------------------------------------------------------

📂 Files Included

• 18.IPL.pbix → Main Power BI Dashboard File

------------------------------------------------------------

📈 Key Insights

• Identified top-performing IPL teams
• Compared team win percentages
• Analyzed player contributions
• Explored venue advantages and match outcomes

▶️ How to Use

1. Download the .pbix file
