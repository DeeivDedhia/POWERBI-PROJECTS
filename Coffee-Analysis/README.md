Coffee Sales Dashboard – Power BI Project

Overview

The Coffee Sales Dashboard is an interactive Power BI project designed to analyze coffee sales performance, customer purchasing trends, revenue growth, and product insights through dynamic visualizations.

Features

• Sales performance analysis
• Product-wise sales insights
• Revenue and profit tracking
• Interactive charts and dashboards
• Dynamic filters and slicers
• Trend analysis and reporting

Tools & Technologies Used

• Power BI
• Power Query
• DAX
• Data Modeling
• Excel / CSV datasets

• Revenue Analysis
• Product Performance
• Sales Trends
• KPI Cards
• Interactive Reports
