Ecommerce Dashboard – Power BI Project

Overview

The Ecommerce Dashboard is a Power BI project developed to analyze online sales performance, customer trends, product insights, and business KPIs through interactive dashboards and visual reports.

Features

• Product performance tracking
• Customer behavior insights
• Interactive charts and filters
• Revenue and profit analysis
• KPI monitoring dashboard
