Call Centre Dashboard – Power BI Project

Overview

The Call Centre Dashboard is an interactive Power BI project designed to analyze call centre performance, customer service metrics, agent productivity, and operational KPIs through dynamic visualizations.

Features

• Call performance analysis
• Agent productivity tracking
• Customer service insights
• Interactive charts and reports
• KPI monitoring dashboard
• Trend analysis and visualization

Tools & Technologies Used

• Power BI
• Power Query
• DAX
• Data Modeling
• Excel / CSV datasets
