HR Analytics Dashboard – Power BI Project

Overview

The HR Analytics Dashboard is a Power BI project developed to analyze employee performance, attrition trends, workforce distribution, and HR KPIs through interactive visualizations and reports.

Features

• Employee attrition analysis
• Workforce performance tracking
• Department-wise insights
• Interactive charts and slicers
• HR KPI monitoring
• Trend analysis and reporting

Tools & Technologies Used

• Power BI
• Power Query
• DAX
• Data Modeling
• Excel / CSV datasets

Dashboard Highlights

• Employee Overview
• Attrition Analysis
• Workforce Insights
• KPI Cards
• Department Performance
• Interactive Reports
