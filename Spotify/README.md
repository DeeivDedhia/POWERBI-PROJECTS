Spotify Dashboard – Power BI Project

Overview

The Spotify Dashboard is an interactive Power BI project designed to analyze music streaming data, artist performance, listener trends, and track insights through dynamic visualizations and KPIs.

Features

• Music streaming analysis
• Artist and track performance insights
• Listener trend visualization
• Interactive charts and slicers
• KPI tracking dashboard
• Trend analysis and reporting

Tools & Technologies Used

• Power BI
• Power Query
• DAX
• Data Modeling
• Excel / CSV datasets

Dashboard Highlights

• Artist Performance Analysis
• Listener Trends
• Track Insights
• KPI Cards
• Interactive Reports
