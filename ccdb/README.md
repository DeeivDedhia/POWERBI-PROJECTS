Credit Card Dashboard – Power BI Project

Overview

The Credit Card Dashboard is an interactive Power BI project designed to analyze credit card transactions, customer spending behavior, financial KPIs, and transaction trends through dynamic visualizations.

Features

• Transaction analysis
• Customer spending insights
• Financial KPI tracking
• Interactive charts and reports
• Dynamic filters and slicers
• Trend analysis dashboard

Tools & Technologies Used

• Power BI
• Power Query
• DAX
• Data Modeling
• Excel / CSV datasets

Dashboard Highlights

• Transaction Overview
• Spending Analysis
• Customer Insights
• KPI Cards
• Financial Trends
• Interactive Reports
