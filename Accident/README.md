Accident Dashboard – Power BI Project

Overview

The Accident Dashboard is a Power BI project developed to analyze road accident data, casualty trends, vehicle involvement, and safety insights through interactive dashboards and visual reports.

Features

• Road accident analysis
• Casualty trend tracking
• Vehicle involvement insights
• Interactive charts and reports
• KPI monitoring dashboard
• Safety trend analysis

Tools & Technologies Used

• Power BI
• Power Query
• DAX
• Data Modeling
• Excel / CSV datasets
