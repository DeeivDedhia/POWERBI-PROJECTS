Uber Analysis Dashboard – Power BI Project

Overview

The Uber Analysis Dashboard is an interactive Power BI project designed to analyze ride-booking trends, trip performance, customer behavior, and operational insights using dynamic visualizations and KPIs.

Features

• Ride and trip analysis
• Booking trend visualization
• Customer and trip insights
• KPI tracking dashboard
• Interactive filters and slicers
• Performance analysis reports

Tools & Technologies Used

• Power BI
• Power Query
• DAX
• Data Modeling
• Excel / CSV datasets

Dashboard Highlights

• Ride Overview
• Trip Performance Analysis
• Booking Trends
• Customer Insights
• KPI Cards
• Interactive Visual Reports
