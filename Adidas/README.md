Adidas Dashboard – Power BI Project

Overview

The Adidas Dashboard is a Power BI project developed to analyze Adidas sales data, product performance, revenue trends, and customer insights through interactive dashboards and visual reports.

Features

• Sales and revenue analysis
• Product performance tracking
• Customer purchase insights
• KPI monitoring dashboard
• Interactive filters and slicers
• Trend analysis and reporting

Tools & Technologies Used

• Power BI
• Power Query
• DAX
• Data Modeling
• Excel / CSV datasets

Dashboard Highlights

• Sales Overview
• Revenue Trends
• Product Insights
• Customer Analysis
• KPI Cards
• Interactive Reports
