Hospital Dashboard – Power BI Project

Overview

The Hospital Dashboard is a Power BI project designed to analyze hospital operations, patient records, healthcare performance, and operational KPIs through interactive dashboards and visual reports.

Features

• Patient data analysis
• Hospital performance tracking
• Healthcare KPI monitoring
• Interactive charts and reports
• Dynamic filters and slicers
• Operational insights dashboard

Tools & Technologies Used

• Power BI
• Power Query
• DAX
• Data Modeling
• Excel / CSV datasets
